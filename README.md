# Iris Dataset Analysis - Task 1

## Objective
The objective of this task is to explore and visualize the Iris dataset using Python. 
This includes understanding the dataset, performing exploratory data analysis (EDA), 
and identifying patterns using visualizations.

## Approach
1. **Data Loading:** Used `pandas` and `seaborn` to load the Iris dataset.  
2. **Dataset Understanding:** Inspected dataset structure with `.shape`, `.columns`, and `.head()`.  
3. **Data Cleaning:** Checked for missing values and ensured dataset is ready for analysis.  
4. **Exploratory Data Analysis (EDA):**  
   - Scatter plots to analyze relationships between variables.  
   - Histograms to examine distributions.  
   - Box plots to detect outliers and visualize spread.  
5. **Visualization Tools:** Used `matplotlib` and `seaborn` for all visualizations.  

## Results & Insights
- Dataset contains 150 rows and 5 columns with three species of iris flowers.  
- Scatter plots revealed relationships between sepal length and width across species.  
- Histograms showed the distribution of petal lengths in the dataset.  
- Box plots highlighted outliers and the spread of values for each species.  
- Overall, the dataset is clean, balanced, and suitable for classification tasks.
