# Iris Dataset Analysis

## Objective
To explore and visualize the Iris dataset and learn data loading, summarization, and visualization techniques using Python.

## Approach
- Load dataset using pandas
- Explore dataset structure (.shape, .columns, .head())
- Create visualizations using matplotlib and seaborn:
  - Scatter plot
  - Histogram
  - Box plot

## Results
- Dataset contains 150 rows and 5 columns with three species.
- Scatter plot shows relationships between sepal length and width.
- Histogram shows distribution of petal length.
- Box plot highlights outliers and spread per species.

## Tools Used
- Python
- Jupyter/Google Colab
- Pandas
- Matplotlib
- Seaborn
